# New Client AFH Build Checklist

Use this checklist to build a new Adult Family Home website from:

- `AFH Website Starter Kit`
- `AFH Automation Engine`

## 1. Client Intake

- [ ] Business name
- [ ] Display name
- [ ] Owner names
- [ ] Address
- [ ] Phone numbers
- [ ] Email
- [ ] Website/domain
- [ ] Logo
- [ ] Photos
- [ ] Services
- [ ] Owner bios
- [ ] Google Business Profile access
- [ ] Form recipient email

## 2. Copy Starter Kit

- [ ] Copy `AFH Website Starter Kit` to a new client folder
- [ ] Rename project folder
- [ ] Update `config/client-config.json`
- [ ] Replace placeholder tokens
- [ ] Replace logo/photos
- [ ] Update social preview image

## 3. Website Customization

- [ ] Home
- [ ] About
- [ ] Services
- [ ] Gallery
- [ ] Contact
- [ ] Schedule Tour
- [ ] Privacy Policy
- [ ] Terms
- [ ] SEO metadata
- [ ] JSON-LD schema
- [ ] Sitemap
- [ ] Robots

## 4. Form Setup

- [ ] Choose Formspree option, or
- [ ] Choose Google Apps Script option
- [ ] Add final form endpoint
- [ ] Test Contact form
- [ ] Test Schedule Tour form

## 5. Automation Setup

- [ ] Create client Google account
- [ ] Create Google Sheet CRM
- [ ] Connect AFH Automation Engine
- [ ] Configure referral agents
- [ ] Configure room availability
- [ ] Set dry-run/test recipient
- [ ] Test weekly referral email
- [ ] Test website lead capture

## 6. Deployment

- [ ] Git init
- [ ] Create GitHub repo
- [ ] Import project into Vercel
- [ ] Connect Namecheap domain
- [ ] Add DNS records
- [ ] Confirm SSL
- [ ] Live test

## 7. Google Setup

- [ ] Google Business Profile
- [ ] Google Search Console
- [ ] Google Analytics
- [ ] Sitemap submission

## 8. Final QA

- [ ] Desktop review
- [ ] Mobile review
- [ ] Hamburger menu
- [ ] Forms
- [ ] Gallery lightbox
- [ ] Map
- [ ] Footer links
- [ ] Social preview
- [ ] Page speed
- [ ] No broken links
- [ ] No horizontal scroll

## 9. Client Review

- [ ] Send website link
- [ ] Ask client to approve content
- [ ] Ask client to approve photos
- [ ] Ask client to approve services
- [ ] Ask client to approve contact info
- [ ] Add final owner photos
- [ ] Complete final edits

## 10. Launch

- [ ] Confirm domain live
- [ ] Submit sitemap
- [ ] Test forms one more time
- [ ] Save credentials and notes
- [ ] Create post-launch support checklist
