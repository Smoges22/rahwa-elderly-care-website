# Deployment

## GitHub

1. Create a new GitHub repository.
2. Commit the customized client site.
3. Push to GitHub.

## Vercel

1. Import the GitHub repository into Vercel.
2. Use default static site settings.
3. Deploy.
4. Test all routes and forms on the Vercel preview URL.

## Namecheap Domain

1. In Vercel, add the client's domain.
2. In Namecheap, open Advanced DNS.
3. Add or update DNS records according to Vercel's instructions.
4. Wait for DNS propagation.
5. Confirm HTTPS is active in Vercel.

## Final Launch Checks

- Homepage loads.
- All interior pages load.
- Mobile navigation works.
- Forms submit.
- Gallery lightbox opens and closes.
- No horizontal scroll on mobile.
- Social preview image loads.
- Sitemap and robots are reachable.
