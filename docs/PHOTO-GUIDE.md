# Photo Guide

Replace placeholder SVG files with real client photography.

Recommended shots:

- Exterior hero photo
- Living room
- Bedroom
- Dining area
- Accessible bathroom
- Outdoor space
- Owner portraits
- Team or caregiving environment

## Naming

Use the existing filenames when possible:

- `assets/images/hero-home.jpg`
- `assets/images/home-exterior.jpg`
- `assets/images/living-room.jpg`
- `assets/images/bedroom.jpg`
- `assets/images/dining-area.jpg`
- `assets/images/accessible-bathroom.jpg`
- `assets/images/outdoor-space.jpg`
- `assets/team/owner-1.jpg`
- `assets/team/owner-2.jpg`

Then update HTML and `styles.css` paths from `.svg` to `.jpg`.

## Privacy

Do not use resident faces or private medical details without written permission.
