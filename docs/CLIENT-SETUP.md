# Client Setup

## 1. Copy the Starter Kit

Copy `AFH Website Starter Kit` into a new client project folder.

Recommended folder name:

```text
client-business-name-afh
```

## 2. Update the Config

Edit `config/client-config.json` with:

- Business name
- Display name
- Tagline
- Website URL
- Address
- Phone numbers
- Email and reply-to email
- Form endpoint
- Google Map embed
- Owner names and bios
- Services and specialties
- Logo and image paths

## 3. Replace Tokens

Search the project for `{{` to find all placeholder tokens.

Common tokens:

- `{{BUSINESS_NAME}}`
- `{{DISPLAY_NAME}}`
- `{{TAGLINE}}`
- `{{WEBSITE_URL}}`
- `{{ADDRESS}}`
- `{{MAIN_PHONE}}`
- `{{ALTERNATE_PHONE}}`
- `{{EMAIL}}`
- `{{FORM_ENDPOINT}}`
- `{{OWNER_1_NAME}}`
- `{{OWNER_1_BIO}}`
- `{{OWNER_2_NAME}}`
- `{{OWNER_2_BIO}}`
- `{{CITY}}`
- `{{STATE}}`

## 4. Final QA

Before launch, test every page, form, image, map, mobile menu, and social preview.
