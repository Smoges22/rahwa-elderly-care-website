# SEO Checklist

Before launch:

- Replace every `{{TOKEN}}`.
- Update page titles.
- Update meta descriptions.
- Update canonical URLs.
- Update Open Graph image paths.
- Replace `assets/social/social-preview.svg` with a final social preview image.
- Update JSON-LD schema on the homepage.
- Update `sitemap.xml`.
- Update `robots.txt`.
- Verify local business NAP consistency.
- Confirm city, state, and service-area copy.
- Check all image alt text.

Recommended schema type:

```json
{
  "@type": "NursingHome"
}
```

Use accurate services and contact information only.
