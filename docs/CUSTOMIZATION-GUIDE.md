# Customization Guide

## Brand

Replace files in `assets/logo/`:

- `logo-horizontal.svg`
- `logo-mark.svg`
- `favicon.svg`

Update colors in `config/client-config.json`, then update CSS variables in `styles.css` if the client needs a custom palette.

## Content

Update:

- Homepage hero headline and trust badges
- About page owner bios
- Services page care offerings
- Gallery captions and alt text
- Contact details
- Legal pages

Keep the tone warm, specific, and family-centered. Avoid unsupported medical claims.

## Design System

The starter keeps the premium design system from the production base:

- Full-image hero
- Premium card components
- Interior page headers
- Mobile hamburger menu
- Gallery lightbox
- Footer action blocks
