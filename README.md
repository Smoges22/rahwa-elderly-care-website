# AFH Website Starter Kit v1.0

Reusable premium Adult Family Home website starter kit based on a completed production AFH design system.

This is the master template for future AFH websites. It keeps the premium layout, mobile navigation, forms, gallery lightbox, SEO structure, JSON-LD schema, sitemap, robots file, and deployment setup while replacing client-specific content with placeholders.

## What Is Included

- Full-image homepage hero
- Premium interior headers
- SaaS-style service and trust cards
- Bright gallery layout with lightbox
- Premium footer sections
- Mobile hamburger navigation
- Contact and Schedule Tour forms
- Google Map embed placeholder
- Social preview setup
- SEO metadata and JSON-LD schema
- `sitemap.xml` and `robots.txt`
- Client configuration file at `config/client-config.json`

## Start a New Client

1. Copy this starter kit folder.
2. Update `config/client-config.json`.
3. Replace placeholders in HTML files.
4. Add real logo and photos to `assets/`.
5. Connect the forms to Formspree or Google Apps Script.
6. Update SEO, schema, sitemap, and robots.
7. Deploy to GitHub and Vercel.

See the docs folder for detailed setup guides.
